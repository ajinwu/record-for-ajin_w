### 2.3 令A,B,C是任意的正规式,证明下述关系成立

1. A|A = A

证明: 设正规式A表示的正规集为L(A)

因为L(A|A) = L(A) U L(A) = L(A)

所以 A|A = A

2. (A*)* = A*

证明: 因为 (A*)* = A** ,并且 A** = A *, 所以(A*)* = A*

3. A* = e| AA*

证明: 因为 AA* = A+, 所以A* = e|AA*

4. (AB)\*A = A(BA)*

证明: 因为(AB)* = e|AB|..., (BA)* = e|BA|...

所以(AB)* = (BA)\*, (AB)\*A = A(BA)*
